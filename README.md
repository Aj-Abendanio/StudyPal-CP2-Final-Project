# StudyPal – Computer Programming 2 Final Project

Authors: Angelika Abendanio, Krisma Rempis, Raffaele Tababan

**StudyPal** is a console based C++ application developed for a final project in **Computer Programming 2**. The program is designed to assist **FEU Tech Computer Science (AI) students** in reviewing important course concepts and testing their understanding through quiz challenges.

The system allows users to select their **year level, semester, course, and module**. After selecting a module, students can read reviewer materials directly in the console and optionally take a **quiz challenge consisting of randomized questions**.

StudyPal also keeps a **record of quiz attempts** which allows users to review their past performance.

## Main Features

- Course and module selection based on **year level and semester**
- **Module reviewer system** that loads study material from text files
- **Quiz challenge system** with randomized questions
- **Quiz scoring and result display**
- **Quiz history tracking** saved to a file
- Organized folder structure for reviewer and quiz content

## Programming Concepts Demonstrated

This project applies the programming concepts covered in **Computer Programming 2**, including:

- Functions and modular programming
- Arrays and dynamic memory usage
- String manipulation
- Structures
- Pointers
- File handling using `fstream`
- Basic Object-Oriented Programming concepts
- Linked data processing for record management

## Purpose

The goal of this project is to demonstrate the practical use of C++ programming concepts by building a simple academic study tool that can help students review course material and assess their understanding through quizzes.
