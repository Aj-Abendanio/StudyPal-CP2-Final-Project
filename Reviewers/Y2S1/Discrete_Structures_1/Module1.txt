Module 2: Introduction to Discrete Structures
1. Introduction
While most computer programming focuses on continuous concepts (like floating-point numbers or fluid animations), Discrete Structures focuses on distinct, separated values. This is the mathematical foundation of Computer Science. It deals with objects that can be counted, such as the bits in a byte, the nodes in a network, or the logical steps in an algorithm. Mastery of discrete mathematics is essential for understanding how computers process logic and manage complex data relationships.
2. Key Concepts

- Propositional Logic: The branch of logic that deals with propositions which can be either True or False. It is the basis for if-else statements in programming. 
- Logical Connectives: Symbols used to combine propositions, such as AND ($\land$), OR ($\lor$), and NOT ($\neg$). 
- Truth Table: A mathematical table used to determine if a compound statement is true or false based on all possible combinations of its inputs. 
- Set Theory: The study of "sets," which are collections of distinct objects. This is fundamental for understanding database queries and unique data collections. 
- Set Operations: Common ways to combine sets, including Union (all elements from both sets) and Intersection (only elements present in both). 
- Universal Quantifiers: Expressions like "For All" ($\forall$) or "There Exists" ($\exists$) used to define the scope of a logical statement.
3. Examples

- Logic in Code: Consider a login system. The logic might be: (Correct_Password AND Valid_MFA). If either is false, the whole statement is false. This is a direct application of Conjunction (AND). 
- Venn Diagrams in Databases: When performing a SQL Inner Join, you are looking for the Intersection of two sets (tables). You only want records where the ID exists in both Table A and Table B. 
- Binary Sets: A computer's file system can be viewed as a set. If you are searching for files that are either .jpg or .png, you are performing a Union operation on those two sets of file types. 
- Negation: In programming, if (!is_admin) uses the NOT operator to reverse the truth value of the variable is_admin.
4. Summary

- Discrete structures deal with countable, distinct elements rather than continuous ranges. 
- Propositional Logic allows us to formalize the decision-making process in software. 
- Truth Tables are tools for visualizing every possible outcome of a logical expression.
