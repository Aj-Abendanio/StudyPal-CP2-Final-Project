Module 15: Image Processing
1. Introduction
While Computer Graphics (Module 14) focuses on creating images from models, Image Processing focuses on analyzing and manipulating existing images. It involves the use of computer algorithms to perform processing on digital images to either improve their quality for human perception or to extract data for machine analysis. This field is the foundation for modern technologies like facial recognition, medical imaging (MRI/CT scans), and autonomous vehicle vision.
2. Key Concepts

- Digital Image: A 2-dimensional array of pixels, where each pixel has a numeric value representing its color or intensity. 
- Point Processing: Operations where each output pixel's value depends only on the corresponding input pixel (e.g., brightness and contrast adjustment). 
- Filtering (Spatial Filtering): Using a small matrix called a Kernel or Mask to change pixel values based on their neighbors. This is used for blurring or sharpening. 
- Histogram: A graphical representation of the distribution of pixel intensities in an image. It helps in understanding the exposure and contrast of a photo. 
- Edge Detection: An image processing technique for finding the boundaries of objects within images (e.g., the Sobel or Canny operators). 
- Image Segmentation: The process of partitioning a digital image into multiple segments (sets of pixels) to simplify or change the representation of an image into something more meaningful.
3. Examples

- Brightness Adjustment: Adding a constant value to every pixel in an image. If a pixel is $(100, 100, 100)$ and you add $20$, it becomes $(120, 120, 120)$, making it brighter. 
- Grayscale Conversion: Converting a color image to black and white by averaging the Red, Green, and Blue channels. 
- Noise Reduction: Applying a "Median Filter" to remove "salt-and-pepper" noise (random white and black pixels) from a grainy security camera feed. 
- Thresholding: Converting a grayscale image into a binary image (pure black and white). This is often used in document scanning to separate text from the background.
4. Summary

- Image processing transforms an input image into an enhanced output image or a set of characteristics. 
- Kernels are the primary tool for effects like blurring, sharpening, and finding edges. 
- Histograms allow us to analyze the balance of light and dark in an image.
