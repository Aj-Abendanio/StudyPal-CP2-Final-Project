MODULE 4: THE COMPUTER AND TASK (ADVANCED INTERACTION)
================================================================================

1. INTERACTION AS INFORMATION TRANSFER
--------------------------------------------------------------------------------

- CORE CONCEPT: Interaction is a process of information transfer between the 
  Human and the Computer to accomplish a specific Task.
- GOALS: When we interact with computers, we have specific goals (e.g., sending 
  a message, calculating a budget). The quality of interaction depends on how 
  efficiently that information moves between the two entities.

2. ABOWD AND BEALE FRAMEWORK (INTERACTION MODEL)
--------------------------------------------------------------------------------

This framework breaks down the interaction into four major components:
- USER: The human with a goal.
- INPUT: How the user communicates the goal to the computer.
- SYSTEM: The computer's internal state and processing.
- OUTPUT: How the computer communicates the result back to the user.



THE CYCLE OF INTERACTION:
1. ARTICULATION: User formulates the goal and performs an input action.
2. PERFORMANCE: The system processes the input and changes its state.
3. PRESENTATION: The system provides a response via output.
4. OBSERVATION: The user perceives and interprets the output to see if the 
   goal was met.

3. INTERACTION DEVICES: INPUT
--------------------------------------------------------------------------------

A. TEXT ENTRY DEVICES:
- QWERTY Keyboard: The standard layout designed to prevent mechanical jams 
  in old typewriters.
- ALPHABETIC Keyboard: Keys arranged A-Z (often slower for trained users).
- CHORD Keyboard: Several keys are pressed at once to enter a character 
  (used in stenography).

B. POSITIONING AND POINTING:
- INDIRECT INPUT: Mouse, Trackball, Touchpad. You move a device to control a 
  separate cursor on the screen.
- DIRECT MANIPULATION: Touchscreens. You interact directly with the objects on 
  the display. No cursor is needed.

C. ADVANCED SENSORS:
- ACCELEROMETERS: Detect tilt and orientation (used in smartphones and 
  gaming controllers like the Wii Remote).
- MULTI-TOUCH: Allows for complex gestures like pinch-to-zoom and swiping.

4. INTERACTION DEVICES: OUTPUT
--------------------------------------------------------------------------------

- VISUAL: LCD and LED screens. 
- ORGANIC LCD (OLCD): Paving the way for flexible and curved displays.
- HAPTIC FEEDBACK: Devices that provide physical sensations (vibrations) to 
  the user to confirm an action.
- AUDITORY: Using sound to alert or provide status updates.

5. NEWER INTERACTION TECHNOLOGIES
--------------------------------------------------------------------------------

- WEARABLE COMPUTING: Devices like smartwatches or fitness trackers that stay 
  with the user constantly.
- SMART SKIN: Experimental interfaces (like "DuoSkin") that turn the user's 
  skin into a touch-sensitive input surface.
- GESTURE RECOGNITION: Controlling systems through body movements without 
  physical contact (e.g., Microsoft Kinect).

6. THE TASK
--------------------------------------------------------------------------------

- Task analysis is the process of learning about ordinary users by observing 
  them in action to understand in detail how they perform their tasks and 
  achieve their intended goals.
- The interface must be designed to support the workflow of the task, not 
  just the features of the software.
